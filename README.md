## Crabbing

App for checking on Crabbing conditions in the bay area

## Deploying to GitHub Pages

This repo is configured to deploy with GitHub Actions using `.github/workflows/deploy.yml`.

1. In GitHub, open **Settings → Pages**.
2. Set **Source** to **GitHub Actions**.
3. Push to `main` to trigger deployment.

Published URL (project site):

- `https://bharris12.github.io/Crabbing/`

Local build check:

```bash
npm run build
```
